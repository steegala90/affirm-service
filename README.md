# Affirm Service

Affirm Service is a modular backend API designed to deliver contextual affirmations through a clean REST interface.

## Vision

To provide structured, category-aware affirmations that can support emotional wellbeing, productivity, and inclusive digital experiences.

## Why This Project

This project demonstrates:
- RESTful API design
- Modular service architecture
- Type-safe backend development (Node + TypeScript)
- Scalable layering (routes → controller → service → model)

## Planned Endpoints (v1)

- GET /health
- GET /affirmations/random
- GET /affirmations?category=confidence
- POST /affirmations
